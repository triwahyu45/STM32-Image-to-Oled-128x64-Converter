# STM32 Image-to-OLED 128x64 Converter

A web-based developer utility that converts images into monochrome byte arrays format for **128x64 OLED displays** running on **STM32** microcontrollers.

🚀 **[Buka Live Image Converter di sini!](https://triwahyu45.github.io/STM32-Image-to-Oled-128x64-Converter/)**

---

## 🌟 Features

*   **Image Processing**: Upload common formats (PNG, JPG) and auto-crop to 128x64 pixels.
*   **Monochrome Thresholding**: Custom slider to adjust thresholding to generate clear black & white pixels.
*   **Hex Code Output**: Generates clean C-code hex arrays formatted for STM32 HAL library or Adafruit SSD1306/u8g2.
*   **Instant Copy**: Single-click button to copy the byte code to your clipboard.\n
